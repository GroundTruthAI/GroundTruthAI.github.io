Objective

To reduce the size of the MNIST Data Set while maintaining its essential characteristics and usability for machine learning and image recognition tasks.

Data Set Description

  • Name:

    MNIST (Modified National Institute of Standards and Technology)

  • Original Size:

    70,000 instances (60,000 training and 10,000 testing), 784 features (28x28 pixel grayscale images), 1 class label (digits 0-9)

  • Classes:

    10 digits (0 to 9)

Dataset Accuracy
Original 97.7%
Distilled 93.7%
